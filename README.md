# Interactive Feedback Form

## 📖 Project Description

This project is a fully interactive customer feedback form built using:

- HTML
- CSS
- JavaScript (Events & Functions)

The form performs real-time validation, highlights fields on hover, and submits data on double-click with confirmation messages.

This project demonstrates DOM manipulation, event handling, and reusable validation logic in JavaScript.

---

## 🚀 Features

✅ Real-time input validation on keypress  
✅ Field highlighting on mouse hover  
✅ Double-click submit functionality  
✅ Reusable JavaScript validation functions  
✅ Clean and responsive UI design  
✅ Proper separation of HTML, CSS, and JS

## ⚙️ How to Run the Project

1. Download or clone the repository.
2. Make sure all three files are in the same folder.
3. Open `index.html` in any modern web browser.
4. Start interacting with the form.

No server setup required.

## 🎯 Functional Overview

### 🔹 Real-Time Validation
- Name must be at least 3 characters.
- Email must follow valid email format.
- Message must contain at least 10 characters.

### 🔹 Events Used
- `keyup` → For live validation
- `mouseover` (via CSS hover) → Field highlight
- `dblclick` → Form submission

### 🔹 JavaScript Concepts Applied
- DOM Manipulation
- Event Listeners
- Regular Expressions
- Reusable Functions
- Conditional Validation Logic
